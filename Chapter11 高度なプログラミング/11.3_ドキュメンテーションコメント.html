11.3_ドキュメンテーションコメント

ドキュメンテーションコメントのフォーマット


<?php
/**
 * 人間に関する情報を扱うクラス
 * 
 * @author Moriaki Kogure
 * @version 0.0.1
 * @license GPL
 */
class Person {
    /** 名前 */
    private string $name;
    /** 名前を取得するメソッド */
    public function getName(): string {
        return $this->name;
    }
}

ドキュメントコメントに使用できる主なタグ

分類 タグ 説明
基本 @author 著作者
@copyright 著作情報
@license ライセンス
@version バージョン
メソッド @param パラメータ
@return 戻り値
@throws 例外
マジックメソッド @method __call/__callStatic呼び出されるメソッド
@property __get/__set軽油で呼び出されるプロパティ
その他 @deprecated 非推奨
@see 参照
