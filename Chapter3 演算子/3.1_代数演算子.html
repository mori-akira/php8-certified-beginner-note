文字に対するインクリメント・デクリメント(マジカルインクリメント)

①

<php
$i = 'Y';
print ++$i;
print ', ';
print $i++;
print ', ';
print $i--;
print ', ';
print --$i;
?>

実行結果


Z, Z, AA, AA

配列同士の演算

①


<pre>
<?php
$ary1 = [1, 2, 3];
$ary2 = [3, 4, 5];
print_r($ary1 + $ary2);
?>
</pre>

実行結果



Array
(
    [0] => 1
    [1] => 2
    [2] => 3
)

②


<pre>
<?php
$assoc1 = [
  'Apple' => 'Red',
  'Orange' => 'Yellow',
  'Melon' => 'Green'
];
$assoc2 = [
  'Grape' => 'Purple',
  'Apple' => 'Green',
  'Strawberry' => 'Red'
];
print_r($assoc1 + $assoc2);
?>
</pre>

実行結果



Array
(
    [Apple] => Red
    [Orange] => Yellow
    [Melon] => Green
    [Grape] => Purple
    [Strawberry] => Red
)
