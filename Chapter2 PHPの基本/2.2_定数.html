2.2_定数

定数の宣言・初期化


const 定数名 = 値

定数名のルール

  1. 先頭に$を付けない
  2. 予約語は使えない
  3. すべて大文字で命名する
  4. 利用できる型は限定される

定数に使用できる型

  • int
  • float
  • string
  • bool
  • null
  • 以上の値を含む配列

define関数による定数定義


define(定数名, 値);

const/defineの相違点

項目 const define
トップレベル以外での宣言 × ○
クラス定数の宣言 ○ ×
変数、関数の戻り値を設定 × ○
実行速度 ○ △

定数に変数を代入


const VALUE = $value; // エラー
define('VALUE', $value) // エラーとならない

事前定義された定数

定数 説明
__FILE__ 実行中のファイル名
__DIR__ 実行中のファイルが存在するディレクトリ名
__LINE__ 実行中の行番号
__FUNCTION__ 実行中の関数名
__CLASS__ 実行中のクラス名
__METHOD__ 実行中のメソッド名
__TRAIT__ 実行中のトレイト名
__NAMESPACE__ 現在の名前空間
DIRECTORY_SEPARATOR フォルダの区切り文字
PATH_SEPARATOR パスの区切り文字
PHP_VERSION 使用しているPHPのバージョン
