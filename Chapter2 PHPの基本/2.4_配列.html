2.4_配列

末尾に要素を追加


<php
$data[] = 'hoge';
$data[] = 'fuga';

配列内容を表示


<pre>
<php
$data = ['hoge', 'fuga'];
print_r($data);
?>
</pre>

実行結果


Array
(
    [0] => hoge
    [1] => fuga
)

連想配列


<?php
$data = [
    'hoge' => 1,
    'fuga' => 2,
];
print $data['hoge'];
?>

一部のみインデックスを指定


<pre>
<?php
$data = [
    'hoge' => 'foo',
    3 => 'bar',
    'piyo'
];
print_r($data);
?>
</pre>

実行結果



Array
(
    [hoge] => foo
    [3] => bar
    [4] => piyo
)

インデックスの重複


<pre>
<?php
$data = [
    'hoge' => 'foo',
    '' => 'bar',
    '' => 'bar2',
    '' => 'bar3',
    'piyo'
];
print_r($data);
?>
</pre>

実行結果



Array
(
    [hoge] => foo
    [] => bar3
    [0] => piyo
)

インデックスに使用できる型

インデックスは整数または文字列を使用できる。 他の型を指定した場合、変換される。 


<pre>
<?php
$data = [
    '' => 'val1',
    true => 'val2',
    0.01 => 'val3',
    false => 'val4',
    null => 'val5',
    '001' => 'val6'
];
print_r($data);
?>
</pre>

実行結果


Deprecated:  Implicit conversion from float 0.01 to int loses precision in C:\xampp\htdocs\selfphp\chap02\array.php on line 74
        

Array
(
    [] => val5
    [1] => val2
    [0] => val4
    [001] => val6
)

なお、インデックスに配列を指定すると、Fatal errorが発生する。
