4.3_ループの制御

外側のループをbreak/continue


<?php
for ($i = 0; $i < 6; $i++) {
    for ($j = 0; $j < 6; $j++) {
        $tmp = $i * 3 + $j; 
        print("{$tmp}&nbsp;");
        if ($tmp > 7) {
            break 2;
        }
    }
}

実行結果


0 1 2 3 4 5 3 4 5 6 7 8

なお、存在する以上の階層を指定した場合、Fatal errorが発生する。

switch文中のcontinue


<?php
for ($i = 0; $i < 3; $i++) {
    for ($j = 0; $j < 3; $j++) {
        $tmp = $i * 3 + $j;
        switch ($tmp % 3) {
            case 1:
            case 2:
                $tmp = $tmp . '--';
                break;
            case 3:
                continue;
        }
        print "{$tmp} ";
    }
}

実行結果


0 1-- 2-- 3 4-- 5-- 6 7-- 8--
